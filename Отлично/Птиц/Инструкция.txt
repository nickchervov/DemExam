Создать гит по ссылке http://10.112.48.9:3000/ и войти по выданным данным. Создать репозиторий Session-1

1. Запустить script
2. Нажать Execute >
3. Обновить базу данных нажав базу данных и обновление
4. Правой кнопкой мыши на Category. Добавить три категории, начиная с ID - 1, 2 , 3. В приложении указаны в следующем порядке (Инвентарь, Одежда, Аксессуары)
5. Картинки в товарах добавить по шаблону /Images/"НАЗВАНИЕ_КАРТИНКИ"."ФОРМАТ_КАРТИНКИ"
6. Добавить в Developer - любое количество производителей и начиная с одного
7. DeveloperPoint - добавить ещё пункты выдачи
8. Users - можно добавить ещё товаров
9. RoleID - не нужно менять

В программе поменять цвета. ProductList изменить ComboBox x:Name="filter", пункты F2,F3,F4 в порядке заполненном в базе данных Category

Ближе к концу экзамена нужно выйти в папку проекта и нажать правой кнопкой мыши GitBush. Ввести коды из блокнота "Коды"
